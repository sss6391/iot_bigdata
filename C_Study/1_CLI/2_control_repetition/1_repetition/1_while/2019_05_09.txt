while ( [상수/변수/표현식] ) {
	_______; 명령문, 명령어 라인=> Statement
}



{ 부터 } 까지 Statement Block 이라고 한다



step1] while () 안에 있는 표현식이 있으면 평가한다

step2] 평가한 결과(표현식)가 참 (1) 이면 while(변수나 상수는 0이 아닌값이면) 이하의 Statement Block을 수행한다
      거짓 (0)이면 statement Block을 빠져나온다


step3] step2에서 참인경우 step1로 이동한다






메타 문자(Meta Characters) 이름(Naming Convention)

opening - closing
() 소괄호: round bracket
{} 중괄호: brace
[] 대괄호: square bracket


\ : Backslash
^ : Caret
$ : Dollar sign
. : Period or dot
| : Vertical bar or Pipe symbol
? : Question mark
* : Asterisk or star
+ : Plus sign
( : Opening round bracket
) : Closing round bracket
[ : Opening square bracket
] : Closing square bracket
{ : Opening brace
} : Closing brace
