-----문자한개 입력------
getchar() 
fgetc(stdin)

-----문자 한개 출력------
putchar( 변수 )
fputc( 변수 ,stdout)


엔터도 입력으로 봄 
\n로 출력됨

문자 EOF는 컨트롤+z 하면 -1



-----문자열 입출력------
puts("문자열" 또는 문자배열)   <= 자동엔터
fputs ("문자열" 또는 문자배열, stdout ) <= 엔터없음


-----문자열 입력------
gets(배열) 
fgets(배열, 사이즈, stdin)  <=자동 널문자추가 사이즈작아짐

엔터도 입력으로 받음

출력시 printf( , 배열)



-----문자열복사-----
strcpy( 복사한 배열 , 처음 배열 )
strncpy( 복사한 배열 , 처음배열, 크기 )

크기가 안맞으면 오류
복사될땐 널문자 복사 안됨 => 무한 출력 오류
